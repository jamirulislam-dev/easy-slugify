# Easy Slugify Monorepo

This repository uses a `pnpm` workspace to develop and maintain the `easy-slugify` package.

## Packages

- **[`easy-slugify`](./packages/easy-slugify)**: A lightweight, fast, and dependency-free TypeScript library to convert any string into a URL-friendly slug.

## Development

### Install Dependencies
To install dependencies for all packages in the workspace:
```bash
pnpm install
```

### Build Packages
To build all packages:
```bash
pnpm build
```

### Run Tests
To run tests across all workspace packages:
```bash
pnpm test
```

For package usage and installation guidelines, please refer to the [easy-slugify Package README](./packages/easy-slugify/README.md).
